mysql role sample
==================

# 概要

データベースの作成とユーザ、パスワードの設定を行います
作成するユーザは作成したDBに関する全ての操作を行えます

1. mysqlのインストール
2. DBの作成
3. DBユーザの作成

# 動作確認済み環境

- ubuntu12.04
- ubuntu14.04

# 設定

    mysql_dbname # 作成するデータベース
    mysql_dbuser # 作成するDBユーザ
    mysql_dbpass # ユーザのパスワード


# 依存関係

依存するroleはありません
